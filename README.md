# gaefrx

Setup
=====

Make sure that the environment variable `GOOGLE_APPENGINE` points to the SDK.


Tests
=====

Use the `test_runner` bash script.


Dev Server
==========

Use the bash script `dev_runner`.

Dependencies
============

* Javascript tooling
  * Node.js
  * npm
  * gulp
  * bower
* Python 2.7+
* Nose
